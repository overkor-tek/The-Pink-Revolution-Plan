## General Questions

### What is the Pink Revolution?

The Pink Revolution is an initiative to transform all blue colors to pink across overkor-tek repositories. It started with simple tab colors and evolved into a comprehensive organizational movement that includes documentation, automation, and consciousness technology infrastructure.

**"One thing became a whole bunch."**

### Why pink?

Pink represents:
- **Bold & Confident** - Standing out from typical tech aesthetics
- **Heart-centered** - Aligned with consciousness work
- **Revolutionary** - Breaking industry norms
- **Unified** - Creating consistent branding across all projects

## Technical Questions

### How do I change blue to pink in CSS?

```css
/* Before */
.button { background-color: #0000FF; }

/* After */
.button { background-color: #FF1493; }
```

See [docs/COLOR_STANDARDS.md](./docs/COLOR_STANDARDS.md) for complete find/replace guide.

---

### How do I change Tailwind classes?

| Replace This | With This |
|--------------|-----------|
| `text-blue-500` | `text-pink-500` |
| `bg-blue-500` | `bg-pink-500` |
| `border-blue-500` | `border-pink-500` |

---

### What about dark mode?

The color palette includes dark mode variants:
- Use `#C71585` (MediumVioletRed) for primary in dark mode
- Use `#FFB6C1` (LightPink) inverted for text on dark backgrounds
- See [docs/COLOR_STANDARDS.md](./docs/COLOR_STANDARDS.md) for CSS variables
