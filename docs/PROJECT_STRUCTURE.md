## File Naming Conventions

### General Rules
- **UPPERCASE** for major documents (README.md, CONTRIBUTING.md)
- **Underscores** for multi-word files (ACTION_CHECKLIST.md)
- **No spaces** in filenames
- **No emojis** in filenames
- **.md** extension for all documentation

### Examples
```

- README.md
- CONTRIBUTING.md
```
